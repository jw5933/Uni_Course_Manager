ABOUT
===================
Welcome.
This is a simple university course management application created with Java and object-oriented methodologies. The main method is located in the SystemDriver class. Currently, downloading a report of the courses does not work (apologies).



===================
HOW TO RUN PROGRAM
===================
after installing JDK

go to the src directory (course_manager/src)
run the following on command line:

javac -Xlint:unchecked SystemDriver.java
java SystemDriver


Once the program is running, it will prompt you to "login".

(case sensitive)admin username: Admin
(case sensitive)admin password: Admin001


You will need to login as the admin first to begin adding students to the system. To select one of the prompted choices, please enter the respective number.
